# Copy Bone Rotations
Copy local rotation of connected bones.  

親子関係のあるボーンのローカル回転をほかのボーンへコピーする。

# How to use
1. Select source bones
1. Push Copy button on properties panel
1. Select target bones
1. Push Paste button

[]()
1. コピーしたいボーンを選択する
1. プロパティパネルにある Copy ボタンを押す
1. ペーストしたいボーンを選択する
1. プロパティパネルにある Paste ボタンを押す
